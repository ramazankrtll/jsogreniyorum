Element Ekleme-Cikartma örneği


 function textForNewElmnt(){
	var metin = document.getElementById("textForNewElement").value;
	var text = document.createTextNode(metin);
	var newElement = document.createElement("P");
	newElement.onclick = function(){deletePFromId(newElement)};
	newElement.appendChild(text);
	var contentOpt = document.getElementById("newElementZone");
	contentOpt.appendChild(newElement);		
}

function deletePFromId(p){
	var element = document.getElementById("newElementZone");
	element.removeChild(p);
}



Yukarıdaki ismi yazan örnekteki işlemımızı adım adım anlatarak açıklayalım.

document.getElementById

bu kod ile textboxımıza verdıgımız id ile bağ kurarak içerisindeki metne ulaşırız.

document.createTextNode(metin)

kodu ile yazılmış olan metni bir düğüm oluşturarak içine atarız.

String olarak gönderilen metinsel değeri sayayı çevirir. Ondalık ifade aynen döndürülür (String to Number).

document.createElement("P");

p tagı altında dugumuzumze bir element ekleriz

newElement.onclick = function(){deletePFromId(newElement)}

yeni eklenen elementin on click özelligine yenı ekleenen elementımızın idsini atarız

newElement.appendChild(text)

metnimizi olusturdugumuz dügümümüze ekleriz.

 function deletePFromId(p){
	var element = document.getElementById("newElementZone");
	element.removeChild(p);
}

on clik özelligiyle üzerine tıklanmış olan yer p olarak idsi gelir ve kaldirma kodumuzla silinmiş olur
